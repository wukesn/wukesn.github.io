### 解题思路

```
遍历s,将左括号压入栈中,反之，弹出栈中最后一个元素，判断它对应的有括号和当前括号是否相等，如果不相等，则返回False，最后查看栈中是否还有元素，若无，则返回True，若有，返回False。
```
### python代码
```
class Solution:
    def isValid(self, s: str) -> bool:
        brackets = {"(":")","{":"}","[":"]"}
        stack = []
        for b in s:
            if b in brackets:
                stack.append(b)
            elif not stack or brackets[stack.pop()] != b:
                return False
        return not stack
```
